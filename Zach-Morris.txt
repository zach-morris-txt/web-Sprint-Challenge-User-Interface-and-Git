1.If you were to describe semantic HTML to the next cohort of students, what would you say?

Some HTML only defines the webpage (without meaning). Semantic gives meaning in addition to that. For example, the paragraph tag declares a paragraph of text and the footer tag declares the foot of the page.

2.Name two big differences between display: block; and display: inline;.

Display block is organized in block format with HxW dimensions. Inline does not break to a new line when used but rather continues off on the same unless directed.

3.What are the 4 areas of the box model?

4.While using flexbox, what axis does the following property work on: align-items: center?

Cross-axis.

5.Explain why git is valuable to a team of developers.

Git allows for multiple editions of code to be edited at the same time by many people and without manipulation of the master code (until directed). This allows several people to build individual projects which then can be unified all without damaging the original. It also displays who did what in case someone creates a large bug.
